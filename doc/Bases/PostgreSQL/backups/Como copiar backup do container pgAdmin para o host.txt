Primeiro, você precisa saber o nome ou o ID do container. Você pode verificar isso com:

```bash
docker ps
```

Após identificar o nome ou o ID do container, você pode usar o seguinte comando para copiar o arquivo para o host:

```bash
docker cp <nome_do_container>:/var/lib/pgadmin/storage/aldenor.oliveira_uol.com.br/bkp-19-10-2024 /caminho/do/host/
```
O Backup deve ser feito como PLAIN
